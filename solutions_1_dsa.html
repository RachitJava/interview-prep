Arrays & Hashing

1. Two Sum 🔥Top
Iterate through array, store `(target - current_val)` in a HashMap. If `target - current_val` is already in the map, you've found the pair indices.
Time: O(N) | Space: O(N)
↗ View Java Solution
2. Best Time to Buy & Sell Stock
Maintain a `min_price` seen so far. At each step, calculate `current_price - min_price` and update `max_profit`.
Time: O(N) | Space: O(1)
↗ View Java Solution
3. Contains Duplicate
Add each element to a HashSet. If `add()` returns false, a duplicate exists.
Time: O(N) | Space: O(N)
↗ View Java Solution
4. Product of Array Except Self 🔥Top
Calculate prefix products in one pass, then suffix products in a second pass. Multiply them to get the result without division.
Time: O(N) | Space: O(1) excluding output array
↗ View Java Solution
5. Maximum Subarray (Kadane's) 🔥Top
Iterate through array, adding elements to `current_sum`. If `current_sum` becomes negative, reset it to 0. Keep track of the `max_sum` seen.
Time: O(N) | Space: O(1)
↗ View Java Solution
6. Group Anagrams
For each string, sort its characters to create a "key". Use a `HashMap>` to group all strings with the same key.
Time: O(N * K log K) where K is max string length | Space: O(N * K)
↗ View Java Solution
7. Top K Frequent Elements
Count frequencies with a Map. Use a PriorityQueue (min-heap of size K) or Bucket Sort to find the top K keys.
Time: O(N log K) or O(N) with bucket sort | Space: O(N)
↗ View Java Solution
8. Encode and Decode Strings
Encode: Join strings with a delimiter like `len#string` (e.g., "5#hello5#world"). Decode: Parse the length, then skip that many characters.
Time: O(N) | Space: O(N)
↗ View Java Solution

Sliding Window

9. Longest Substring Without Repeating 🔥Top
Use two pointers (L, R) and a HashSet. If `char[R]` is in set, remove `char[L]` and move L until `char[R]` can be added. Store max `R-L+1`.
Time: O(N) | Space: O(min(m, n)) where m is charset size
↗ View Java Solution
10. Minimum Window Substring
Sliding window + Frequency Map. Expand R until window contains all target chars, then shrink L as much as possible while maintaining validity.
Time: O(N + M) | Space: O(M) where M is target string len
↗ View Java Solution
11. Sliding Window Maximum
Use a Deque to store indices of potential maxima. Ensure elements in deque are in decreasing order of their array values. Max for current window is at `peekFirst()`.
Time: O(N) | Space: O(K) where K is window size
↗ View Java Solution
12. Longest Repeating Character Replacement
Sliding window. Track max frequency of any char in current window. If `(window_len - max_freq) > k`, shrink from left. Window size is the answer.
Time: O(N) | Space: O(26) for uppercase alphabet
↗ View Java Solution

Two Pointers & Binary Search

13. 3Sum 🔥Top
Sort array. Iterate with `i`, use Two Pointers (L, R) on remaining part to find `nums[i] + nums[L] + nums[R] == 0`. Skip duplicates for all three indices.
Time: O(N²) | Space: O(1) or O(N) for sort
↗ View Java Solution
14. Container With Most Water
Two pointers at L=0 and R=len-1. Calculate `min(H[L], H[R]) * (R-L)`. Move the pointer that points to the SHORTER height.
Time: O(N) | Space: O(1)
↗ View Java Solution
15. Trapping Rain Water 🔥Top
Two pointers at L and R. Track `leftMax` and `rightMax`. At each step, process the side with the smaller max height and add `max - current` to total.
Time: O(N) | Space: O(1)
↗ View Java Solution
16. Search in Rotated Sorted Array 🔥Top
Modified Binary Search. Determine which half is "sorted". If target lies within the sorted half, search there; otherwise search the other half.
Time: O(log N) | Space: O(1)
↗ View Java Solution
17. Find Minimum in Rotated Array
Binary search. If `nums[mid] > nums[right]`, then minimum is in the right half. Otherwise, minimum is in the left half (including mid).
Time: O(log N) | Space: O(1)
↗ View Java Solution
18. Median of Two Sorted Arrays
Binary search on partitions of the smaller array. Ensure left side of partitions has same total count as right side and `maxLeft1 <= minRight2` and vice versa.
Time: O(log(min(N,M))) | Space: O(1)
↗ View Java Solution

Linked List

19. Reverse Linked List 🔥Top
Iterative: Use `prev`, `curr`, `next`. `curr.next = prev`, then shift all three forward. Recursive: Head recursion, then set `head.next.next = head`.
Time: O(N) | Space: O(1) iterative
↗ View Java Solution
20. Merge Two Sorted Lists
Iterative: Use dummy node and tail pointer. Compare heads, attach smaller, move pointers. Recursive: `if(L1.val < L2.val) L1.next = merge(L1.next, L2); return L1;`
Time: O(M+N) | Space: O(1) iterative
↗ View Java Solution
21. Detect Cycle 🔥Top
Floyd's Tortoise and Hare. `slow` moves 1 step, `fast` moves 2 steps. If they meet, there is a cycle. If `fast` reaches null, no cycle.
Time: O(N) | Space: O(1)
↗ View Java Solution
22. LRU Cache 🔥Top
`HashMap` + Doubly Linked List. Map gives O(1) access; DLL maintains order of use. `put/get` moves node to front; evict from tail.
Time: O(1) for all ops | Space: O(Capacity)
↗ View Java Solution
23. Merge K Sorted Lists
Use a PriorityQueue (min-heap) of size K. Add heads of all K lists. Poll smallest, attach to result, and add that polled node's `next` to heap.
Time: O(N log K) | Space: O(K)
↗ View Java Solution

Trees & Graphs

24. Invert Binary Tree
Recursively `swap(root.left, root.right)`. `temp = left; left = invert(right); right = invert(temp);`
Time: O(N) | Space: O(H) recursion stack
↗ View Java Solution
25. Max Depth of Binary Tree
`1 + max(depth(left), depth(right))`. Standard DFS traversal.
Time: O(N) | Space: O(H)
↗ View Java Solution
26. Validate BST 🔥Top
Recursion with range `(min, max)`. `isValid(node, min, max) = node.val > min && node.val < max && leftValid && rightValid`.
Time: O(N) | Space: O(H)
↗ View Java Solution
27. Level Order Traversal (BFS)
Use a Queue. Poll parent, add to level list, offer its non-null children to queue for next level.
Time: O(N) | Space: O(W) max level width
↗ View Java Solution
28. Lowest Common Ancestor 🔥Top
If node is p or q, return node. Search left and right. If left and right both return non-null, current node is LCA. Otherwise return the non-null result.
Time: O(N) | Space: O(H)
↗ View Java Solution
29. Serialize/Deserialize Binary Tree
Serialize: Preorder traversal, use "X" for null and "," delimiter. Deserialize: Use a Queue/Pointer to rebuild via preorder recursion.
Time: O(N) | Space: O(N)
↗ View Java Solution
30. Number of Islands 🔥Top

Time: O(V+E) | Space: O(V)
↗ View Java Solution

Dynamic Programming

34. Climbing Stairs
Basically Fibonacci. `dp[i] = dp[i-1] + dp[i-2]`. Can optimize to O(1) space with two variables.
Time: O(N) | Space: O(1) optimized
↗ View Java Solution
35. Coin Change 🔥Top
`dp[i] = min(dp[i], 1 + dp[i-coin])`. Fill DP array from 1 to target amount. Initialize with infinity.
Time: O(Amount * N_coins) | Space: O(Amount)
↗ View Java Solution
36. Longest Increasing Subsequence 🔥Top
`dp[i] = max(dp[j] + 1)` for all `j < i` where `nums[j] < nums[i]`. Can be solved in O(N log N) using patience sorting/binary search.
Time: O(N²) or O(N log N) | Space: O(N)
↗ View Java Solution
37. House Robber
`dp[i] = max(dp[i-1], nums[i] + dp[i-2])`. Max of (don't rob current, rob current + total from 2 houses back).
Time: O(N) | Space: O(1) optimized
↗ View Java Solution
38. Word Break
`dp[i]` is true if `dp[j]` is true AND `s.substring(j, i)` is in dictionary.
Time: O(N²) | Space: O(N)
↗ View Java Solution
39. Longest Common Subsequence
2D DP. `if(s1[i] == s2[j]) dp[i][j] = 1 + dp[i-1][j-1]; else dp[i][j] = max(dp[i-1][j], dp[i][j-1]);`
Time: O(M*N) | Space: O(M*N)
↗ View Java Solution
40. 0/1 Knapsack 🔥Top
`dp[w] = max(dp[w], val[i] + dp[w-wt[i]])`. Use 1D DP array filled backward to save space and avoid using same item twice.
Time: O(N*W) | Space: O(W) weight limit
↗ View Java Solution
41. Edit Distance
`if(s1[i] == s2[j]) return dp[i-1][j-1]; else return 1 + min(insert, delete, replace)`. 2D DP matrix.
Time: O(M*N) | Space: O(M*N)
↗ View Java Solution

↗ View Java Solution
43. Min Stack
Use two stacks (standard stack + `minStack`). On push, add `min(val, minStack.peek())` to `minStack`.
Time: O(1) for all | Space: O(N)
↗ View Java Solution
44. Kth Largest Element 🔥Top
Min-heap of size K: O(N log K). QuickSelect: O(N) average. Binary search on value range: O(N log Range).
Time: O(N log K) or O(N) | Space: O(K)
↗ View Java Solution
45. Find Median from Data Stream
Two heaps: `maxHeap` (left half) and `minHeap` (right half). Rebalance to ensure `maxHeap.size >= minHeap.size` and diff <= 1. Median is `maxHeap.peek()` or `avg(peeks)`.
Time: O(log N) for add | Space: O(N)
↗ View Java Solution

Time: O(M*N * 3^L) | Space: O(Total_Words_Chars)

Backtracking & Intervals

48. Combination Sum
Standard backtracking. `backtrack(idx, currentTarget, list)`. At each step, either take `nums[idx]` and keep idx (to allow reuse) or skip to `idx+1`.
Time: O(2^T) where T is target value | Space: O(T/min_val)
↗ View Java Solution
49. Merge Intervals 🔥Top
Sort by start time. Iterate: if `curr.start <= prev.end`, merge by updating `prev.end = max(prev.end, curr.end)`. Else add new interval.
Time: O(N log N) | Space: O(N) or O(log N) for sort
↗ View Java Solution

Time: O(N!) | Space: O(N)
