🤖 Advanced AI & Model Architecture

1. Explain the Transformer Attention mechanism simply. It allows the model to "focus" on relevant words regardless of distance. Example: In "The bank of the river," the model pays 'attention' to "river" to understand "bank" refers to land, not a building.
2. What is Hallucination? How do you prevent it? When an LLM generates confident but false info. Prevention: 1. RAG (contextual ground truth). 2. Lower Temperature. 3. Few-Shot Prompting with correct examples.
3. Precision vs Recall? Precision = "Of all I labeled positive, how many were actually positive?" Recall = "Of all that WERE positive, how many did I find?" In SmartCPR, Recall is more important (don't miss a cardiac arrest).
4. Overfitting: How to detect and fix? Detect: Training error is low, but Validation error is high. Fix: More data, Dropout layers, Regularization (L1/L2), or simplifying the model.

🛡️ Security, JWT & OAuth

5. How does JWT (JSON Web Token) Auth work? 1. User logs in. 2. Server generates a signed Token. 3. Client sends this in the 'Authorization' header for every request. 4. Server verifies the signature without hitting the DB.
6. Difference between OAuth 2.0 and OpenID Connect (OIDC)? OAuth 2.0 is for Authorization (accessing resources). OIDC is a layer on top of OAuth for Authentication (proving who the person is).
7. What is CSRF and how to prevent it? Cross-Site Request Forgery. An attacker tricks a user's browser into sending a request to your app. Fix: Use CSRF Tokens or `SameSite=Strict` cookies.

⚡ Quick-Fire Revision (Secondary Topics)

• CAP Theorem: Choose 2: Consistency, Availability, Partition Tolerance. Most web apps pick AP.
• SQL Index: B-Tree by default. Speeds up SELECT, slows down INSERT/DELETE.
• REST vs gRPC: REST (JSON, HTTP/1) is for external. gRPC (Binary, HTTP/2) is 10x faster for internal.
• Spring Beans: Singleton by default (one per container).
• Java Streams: map (one-to-one), filter (choose some), collect (reduce to list).
• Docker vs VM: Docker shares the OS kernel (lightweight). VM has its own OS (heavy).
• Microservice Auth: Authenticate at the API Gateway, then pass JWT to internal services.
• Kubernetes Liveness: Restarts the container if it crashes. Readiness: Stops traffic if it's slow/booting.
