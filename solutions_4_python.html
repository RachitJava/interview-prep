🐍 Solutions 4: Python (50 Topics)

Core Python

1. List vs Tuple vs Set vs Dictionary 🔥Top
List (Mutable, Ordered), Tuple (Immutable, Ordered), Set (Mutable, Unordered, Unique), Dictionary (Key-Value pairs, Ordered by insertion in 3.7+).
2. What are *args and **kwargs? 🔥Top
*args: Variable number of non-keyword arguments (received as a tuple). **kwargs: Variable number of keyword arguments (received as a dictionary).
3. What is GIL (Global Interpreter Lock)? 🔥Top
A mutex that allows only one thread to control the Python interpreter. Prevents multiple native threads from executing Python bytecodes at once. Affects multi-core CPU-bound performance.
4. Mutable vs Immutable objects in Python
Mutable: Can change after creation (List, Dict, Set). Immutable: Cannot change (Int, Float, Str, Tuple). Changing an immutable object creates a NEW copy.
5. List comprehension vs generator expression
List Comp `[]`: Returns a full list immediately. Generator `()`: Returns an iterator that yields one value at a time (saves memory).
6. Deep copy vs shallow copy 🔥Top
Shallow: Copies reference of nested objects. Deep: Recursively copies everything (nested objects are independent).
7. What are decorators? 🔥Top
Functions that wrap another function to modify its behavior without changing source code. Used for logging, authentication, and timing.
8. What is a closure in Python?
A technique where a nested function remembers the values from its enclosing scope even after the outer function has finished executing.
9. Lambda functions — when to use?
One-line anonymous functions. Use for simple, short transformations, often inside `map()`, `filter()`, or `sort()`.
10. What does "if __name__ == '__main__'" do?
Ensures that code inside the block only runs if the script is executed directly, not when imported as a module in another script.

OOP in Python

11. @staticmethod vs @classmethod vs instance method 🔥Top
Instance (needs `self`), Class (needs `cls`, can access class state), Static (utility only, no `self`/`cls`).
12. What is __init__ vs __new__?
__new__: Creates the actual instance (constructor). __init__: Initializes the instance (initializer). `__new__` is rarely overridden except for Singeltons or immutable types.
13. What is MRO (Method Resolution Order)?
The order in which Python searches for a method in a multiple-inheritance hierarchy. Uses C3 Linearization. View using `ClassName.mro()`.
14. Multiple inheritance — diamond problem?
When a class inherits from two parents who have a common ancestor. Python's MRO resolves this reliably by defining a strict search order.
15. What are dunder/magic methods?
Methods starting and ending with double underscores. `__str__` (readable), `__repr__` (unambiguous), `__len__`, `__eq__`.
16. Abstract base classes (ABC) in Python
Force subclasses to implement specific methods. Defined in the `abc` module using the `@abstractmethod` decorator.
17. Property decorator — getter/setter
Allows a method to be accessed like an attribute. Use `@property` for getters and `@name.setter` for setters to encapsulate data.
18. What is monkey patching?
Changing or extending code at runtime. Example: Redefining a function of a library at execution time for testing purposes.

Advanced Python

19. What are generators? yield vs return 🔥Top
return exits function. yield pauses function, saving its state to continue later. Generators are memory efficient for big data.
20. What are context managers? (with statement) 🔥Top
Handle resource setup/cleanup. `with open() as f`. Implemented via `__enter__` and `__exit__`.
21. asyncio — async/await 🔥Top
Single-threaded event loop for handling high-concurrency I/O bound tasks. Excellent for network calls and fast APIs.
22. Multithreading vs Multiprocessing in Python
Threading (Shared memory, hit by GIL, good for I/O). Multiprocessing (Separate memory, bypasses GIL, good for CPU-bound tasks).
23. What is a metaclass?
A class that creates another class. "Classes are instances of metaclasses". Typical base is `type`.
24. Python memory management — reference counting
Python counts how many references point to an object. When it hits 0, the object is immediately deleted. Also uses Generational GC for cycles.
25. What are slots in Python?
`__slots__` tells Python not to use a dynamic `__dict__` for each instance. Saves massive memory when creating millions of objects.
26. Type hints — why and how to use them
Annotations like `def foo(x: int) -> str:`. Improves IDE support, readability, and allows static analysis (e.g. `mypy`).
27. Dataclasses — @dataclass decorator
Auto-generates `__init__`, `__repr__`, and `__eq__` for data-only classes. Added in 3.7.
28. What is pip vs conda vs poetry?
pip (standard package manager), conda (environment + science packages), poetry (modern dependency management and packaging).

Python for Data/AI

29. NumPy — arrays, broadcasting 🔥Top
Broadcasting: How NumPy treats arrays of different shapes during arithmetic. Small arrays are "stretched" to match the shape of larger arrays.
30. Pandas — DataFrame operations
Provides tabular data structures. Key ops: `groupby`, `merge` (joining), `pivot_table`, and `vectorized` string operations.
31. How to read a 10GB CSV file in Python?
Use Chunking in Pandas: `pd.read_csv('file.csv', chunksize=100000)`. Process one page at a time.
32. What is pickle? Security concerns?
Serialization for Python objects. Danger: De-pickling can execute arbitrary code. Never unpickle data from untrusted sources.
33. Virtual environments
Isolates project dependencies (`venv`). Ensures Project A doesn't break Project B when they need different library versions.
34. Python's import system — __init__.py
`__init__.py` makes a directory a package. Can also be used for package-level initialization and exposing specific members.

Flask/FastAPI

35. Flask vs FastAPI 🔥Top
Flask: Synchronous, mature, simple. FastAPI: Asynchronous (async code), uses Pydantic for validation, 10x faster performance.
36. How does Flask routing work?
Uses the `@app.route('/')` decorator to map a URL to a specific Python function.
37. What is WSGI vs ASGI? 🔥Top
WSGI: Synchronous standard for Flask/Django. ASGI: Asynchronous successor for FastAPI, supports WebSockets and HTTP/2.
38. How to handle authentication in Flask?
Use **Flask-Login** for sessions or **Flask-JWT-Extended** for token-based auth.
39. Blueprint in Flask
Organizes a large app into smaller, reusable sub-components (modules) like `user_bp`, `admin_bp`.
40. FastAPI — Pydantic models
Pydantic performs data validation and serialization. If a request body doesn't match the model schema, FastAPI returns an error automatically.

Python Coding Problems

41. Reverse a string without slicing
`"".join(reversed(text))` or using a loop to build a new string from back to front.
42. Find duplicates in a list — O(n)
`seen = set(); dups = [x for x in nums if x in seen or seen.add(x)]`. One pass through list.
43. Flatten a nested list
Recursive function: `for sub in mylist: if isinstance(sub, list): flatten(sub) else: out.append(sub)`.
44. Implement a LRU cache using OrderedDict
Check if key in dict; if yes, move to end (MRU). If over capacity, pop item with `last=False`.
45. Merge two sorted lists
Standard two-pointer approach, comparing heads and building a result list.
46. Check if string is palindrome
`s == s[::-1]` (slicing) or two pointers comparing outer characters moving inward.
47. Find most frequent element
`collections.Counter(mylist).most_common(1)` or using a simple dictionary to count.
48. Implement binary search
Classic `while low <= high: mid = (low+high)//2; check nums[mid]`.
49. Write a thread-safe singleton
Using a **Lock** around instance creation inside the `__new__` method or a MetaClass.
50. Implement rate limiter decorator 🔥Top
Use `time.time()` inside a decorator to track last call timestamp. If `now - last < interval`, raise Exception or return 429.
